# Get Started
1.  Install Docker on Windows,Ubuntu or Mac 
2.  Clone the repo 
3.  ``docker build -t [NAME]:[TAG] .`` 
4. ``docker run -it -d --name [CONTAINER_NAME] -p 3000:3000 [IMAGE]:[TAG]``
5.  Got localhost:3000 to view your application.
